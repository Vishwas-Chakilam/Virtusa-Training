Day 13: Java 8 Features (Functional Programming)
    
    Huge shift.
    - Lambdas: (args) -> { body } 
      Makes code way cleaner for anonymous classes.
    - Functional Interfaces: Interface w/ EXACTLY ONE abstract method. (Predicate, Consumer, Function, Supplier).
    
    - Streams API: pipeline ops on collections. filter(), map(), etc.
      Remember streams are LAZY. Terminal op (like collect or forEach) is needed to actually run them.
      
    - Optional<T>: try to avoid returning null to fix NullPointerExceptions.
    
    // Mind blown by streams today. way less boilerplate than traditional for-loops.
