# Day 15: Inner Classes

## What are Inner Classes?
In Java, it is possible to nest classes (a class within a class). The purpose of nested classes is to group classes that belong together, making your code more readable and maintainable.

## Types of Inner Classes
1. **Nested Inner Class (Non-static)**: Can access all members of outer class including private.
2. **Static Nested Class**: Cannot access non-static members of outer class.
3. **Method Local Inner Class**: Created inside a method.
4. **Anonymous Inner Class**: A class without a name, created and instantiated in a single statement. Primarily used for overriding methods of interfaces or abstract classes on the fly.
