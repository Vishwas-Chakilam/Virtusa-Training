Day 3: Static & Access Modifiers
    
    - static keyword means belongs to the Class, not the object. 
      (can't use 'this' in a static method!!)
      
    Access Modifiers:
    1. private: only inside same class
    2. default: package level (no keyword)
    3. protected: package + child classes (even in diff packages)
    4. public: everywhere
    
    final keyword = constant. final class cannot be extended. String is final.
