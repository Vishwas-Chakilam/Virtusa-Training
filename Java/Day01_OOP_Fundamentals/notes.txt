Day 1: OOP Basics
    
    * Encapsulation: Hiding data w/ private vars + getters/setters. Good practice.
    * Inheritance: IS-A relationship. extends keyword. 
      Note to self: Java doesn't support multiple inheritance w/ classes (diamond problem), only interfaces.
    * Polymorphism: Many forms. 
      - Method Overloading (compile time) - same name, diff args.
      - Method Overriding (runtime) - child provides specific implementation.
    * Abstraction: abstract classes & interfaces. 
      Interfaces are basically contracts.
