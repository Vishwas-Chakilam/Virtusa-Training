# Day 40: Flow Control

## If...Else Statements
Uses the keywords `if`, `elif`, and `else`. Conditions are followed by a colon `:` and the block is indented.
```python
if a > b:
    print("A is greater")
elif a == b:
    print("Equal")
else:
    print("B is greater")
```

## While Loops
Executes block as long as condition is true.
```python
while i < 10:
    i += 1
```

## For Loops
Iterates over a sequence (list, tuple, dictionary, set, or string).
```python
for item in my_list:
    pass
```

## Control Flow Tools
- `break`: stop the loop completely.
- `continue`: stop current iteration, continue with the next.
- `pass`: a null operation (placeholder).
