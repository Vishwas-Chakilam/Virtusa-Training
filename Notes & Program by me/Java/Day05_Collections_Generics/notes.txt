Day 5: Collections Framework
    
    Finally past plain arrays. Collections are dynamic.
    
    - List: ordered, allows duplicates. (ArrayList is best for read, LinkedList for insert/delete in middle).
    - Set: no duplicates. (HashSet = no order but O(1) ops, TreeSet = sorted).
    - Map: Key-Value pairs. Keys must be unique. (HashMap mostly used).
    
    Generics: <T>. provides compile-time type safety so we don't have to cast everything from Object and get ClassCastException later.
