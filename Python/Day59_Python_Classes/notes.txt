# Day 59: Python Classes

## Basics of Objects
Define a class using `class` keyword.
`__init__(self)` is the initializer/constructor.
`self` maps to the new object instance created, similar to `this` in Java.

## Instance vs Class Attributes
- **Instance Attributes**: Declared under `__init__`, unique to each object.
- **Class Attributes**: Declared loosely inside the class, shared globally among all instances of that class.
