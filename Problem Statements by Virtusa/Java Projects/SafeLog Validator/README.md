# Java Project: SafeLog Validator

## Problem Statement
Corporate environments need a secure and reliable way to validate employee passwords against strict security policies.

## Objectives
- Enforce standard password complexity rules
- Provide granular feedback to users on security gaps

## Key Features
- Minimum length validation (8 characters)
- Case-sensitive checks (Uppercase letters)
- Numerical requirement (At least one digit)
- Interactive session with retry mechanism

## Technology Suggestions
- Core Java (String manipulation, Loops)
- Character utility methods

## Expected Outcome
A modular utility that helps users create stronger passwords through real-time feedback.
