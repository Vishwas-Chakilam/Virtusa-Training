Day 12: Enums, Annotations, I/O
    
    Enums: type-safe constants. they are actually classes extending java.lang.Enum under the hood.
    
    Annotations: metadata like @Override. Retention policies = SOURCE, CLASS, RUNTIME.
    
    I/O Streams:
    Byte Streams (InputStream/OutputStream) for raw binary, Character Streams (Reader/Writer) for text.
    * Use Buffered streams (BufferedReader) for better performance so we don't hit the disk on every single byte.
    
    Serialization = object -> byte stream. Must implement Serializable interface.
    Use 'transient' keyword to skip fields (like passwords).
