Day 4: Arrays & Strings
    
    Arrays: fixed length, contiguous memory. int[] nums = new int[5];
    
    Strings are IMMUTABLE. 
    - String pool is a special memory area. 
    - "hello" == "hello" is true bc they point to same pool instance.
    - new String("hello") creates brand new object on heap.
    
    Use StringBuilder class if doing lots of appends inside a loop, otherwise memory kills the app.
