# Day 39: Getting Started

## Variables and Types
Variables are created the moment you first assign a value to it.
- **Numbers**: `int` (5), `float` (5.0), `complex` (1j)
- **Strings**: `'hello'` or `"hello"`
- **Booleans**: `True` or `False`

## Type Checker
Use the `type()` function to know the type of any object.

## Basic Arithmetic
- `+` (Addition), `-` (Subtraction)
- `*` (Multiplication), `/` (Division - returns float)
- `//` (Floor Division - returns int)
- `%` (Modulus), `**` (Exponentiation)
