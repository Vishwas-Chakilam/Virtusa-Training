# Day 27: SQL Basics

## SQL Sublanguages
- **Data Definition Language (DDL)**: Used to define the database schema.
  - Commands: `CREATE`, `ALTER`, `DROP`, `TRUNCATE`
- **Data Manipulation Language (DML)**: Used to add, retrieve or update the data.
  - Commands: `INSERT`, `UPDATE`, `DELETE`
- **Data Query Language (DQL)**: Retrieve data from the DB.
  - Commands: `SELECT`
- **Data Control Language (DCL)**: Control access to data.
  - Commands: `GRANT`, `REVOKE`
- **Transaction Control Language (TCL)**: Manage the transactions.
  - Commands: `COMMIT`, `ROLLBACK`, `SAVEPOINT`
